<h2 align="center" id="about-the-game">ğŸ§  About The Game</h2>

<p align="center">
Logic Rush is a console-based C++ math challenge game where every move tests your logic and problem-solving skills. <br>
Players start at the top-left cell <b>(0,0)</b> and aim to reach the final cell <b>(N-1, N-1)</b>.  
Each cell contains a math problem that must be solved correctly to enter.  
Correct answers allow progress and earn points, while mistakes cost lives.
</p>

<p align="center">
ğŸ“Š The game includes three difficulty levels:<br><br>

<b>ğŸŸ¢ Easy:</b> 4Ã—4 grid, addition and subtraction, 4 lives<br>
<b>ğŸŸ¡ Normal:</b> 5Ã—5 grid, addition, subtraction, multiplication, integer division, 3 lives, bonus + extra life cells<br>
<b>ğŸ”´ Hard:</b> 7Ã—7 grid, mixed expressions, squares, modulo operations, 2 lives, trap cells<br><br>

Each level introduces unique challenges to keep the gameplay engaging and replayable.
</p>

## How to Build and Play
From the project folder:
```bash
# Linux/macOS
g++ -std=c++17 main.cpp Game.cpp Problems.cpp -o main
./main

# Windows (MinGW)
g++ -std=c++17 main.cpp Game.cpp Problems.cpp -o main.exe
main.exe
```
If `g++` is missing, install a C++ compiler (Ubuntu/Debian: `sudo apt install build-essential`; Windows: install MinGW-w64 and use its `g++`).
